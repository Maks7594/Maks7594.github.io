How to Join

There are a few ways to join my server.

Java Edition

1. Open Minecraft.
2. Go to Multiplayer.
3. Go to Direct Connection (A) or Add Server. (B)
3A. Paste in untitledsky.minehut.gg in the Server Address textbox and hit Join Server.
3B. Name the server whatever you'd like, paste in untitledsky.minehut.gg in Server Address. Then, hit Done and double-click on our server.

You also can join minehut.gg and enter command /join UntitledSky.

Bedrock Edition

1. Open Minecraft.
2. Click Play.
3. Go to Servers.
4. Scroll down and click "Add Server".
5. In Server Address, paste in bedrock.minehut.com and port 19132 and join. Then, enter command /join UntitledSky. Homepage
